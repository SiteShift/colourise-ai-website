## INTERNAL LINKING RULES

### Link Hierarchy

```
Level 1: Homepage
    ↓ (links to all pillars)
Level 2: Pillar Pages
    ↓ (links to all clusters in topic)
Level 3: Cluster Articles
    ↓ (links back to pillar + related clusters)
Level 4: Tool Pages
    ← (linked from relevant articles)
```

### Linking Requirements Per Page

| Page Type | Links OUT | Links IN (Target) |
|-----------|-----------|-------------------|
| Homepage | 5 pillars + featured articles | All pages |
| Pillar Page | All clusters (10-15) + 2-3 other pillars | 15+ from clusters |
| Cluster Article | 1 pillar + 3-4 related clusters + 1 tool | 5+ from other clusters |
| Tool Page | 1 pillar + 2-3 related articles | 10+ from articles |

### Anchor Text Strategy

| Link Type | Anchor Text Style | Example |
|-----------|-------------------|---------|
| To Pillar | Exact/partial match | "complete guide to AI colorization" |
| To Cluster | Natural phrase | "learn how AI colorization works" |
| To Tool | Action-oriented | "try our free color palette extractor" |
| To Comparison | Question-based | "which colorization app is best?" |

## CONTENT TEMPLATES

### Pillar Page Template
```
1. Hero Section (H1 + intro)
2. Table of Contents
3. Quick Answer Box (featured snippet target)
4. Main Content (H2s for each major section)
5. Embedded Tool/Calculator (if applicable)
6. Cluster Links Section ("Explore This Topic")
7. FAQ Section (schema markup)
8. Related Pillars Section
9. CTA (Waitlist/Download)
```

### Cluster Article Template
```
1. H1 + Meta Description
2. Breadcrumb (Home > Pillar > Article)
3. Quick Answer (first 100 words)
4. Table of Contents
5. Main Content (H2/H3 hierarchy)
6. Practical Examples/Images
7. Step-by-Step Section (if applicable)
8. Tool Embed (if applicable)
9. FAQ (3-5 questions)
10. Related Articles (3-4 links)
11. Back to Pillar CTA
12. Main CTA (Waitlist)
```

### Tool Page Template
```
1. H1 + Description
2. Tool Interface
3. How to Use (3-5 steps)
4. Use Cases
5. Related Articles (3-5)
6. FAQ (3 questions)
7. CTA (Waitlist)
```
